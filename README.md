五种上火症状

1、心火：分虚实两种，症状有舌尖发红，口干舌燥，反复口腔溃疡、牙龈肿痛，睡眠质量差总是做噩梦。同时伴有心烦意乱，手脚心发热，盗汗，小便发黄发热，大便干结。

2、肺火：主要表现为脸上长有疖子，咽鼻喉疼痛干燥，咳嗽时胸口疼痛，干咳无痰，痰中带血或者痰液发黄而黏稠，睡眠质量差，潮热盗汗，舌头前半部位发红等。

3、胃火：分虚实两种，主要症状有口渴口干口臭，轻微咳嗽，牙齿疼痛，牙龈出血和肿胀，便秘腹胀。同时伴有烦热胃痛，喜欢喝冷水，容易感觉到饥饿，整个舌头黄腻，大便干硬等。

4、肝火：我们常称一些情绪容易激动的人为“肝火大”。当有肝火时，舌质发红，尤其是集中在舌头两侧，脾气暴躁，失眠多梦，头痛头晕脸红眼赤，同时伴有耳聋耳鸣，口干口苦，舌苔发黄，舌苔增厚以及大便干结等。

5、肾火：肾阴不足时主要表现有舌头发红，舌苔少，腰膝酸软，头晕目眩，形体消瘦。同时伴有口干舌燥耳鸣，牙齿脱落或者睡眠质量不好。阴虚火旺时会出现心烦意乱、盗汗以及手脚心发热，严重的话舌头发红，没有苔或少苔。
